# Self-Balancing Robot PID Control

Academic control systems project focused on the modeling and PID control of a two-wheeled self-balancing robot based on the LEGO Mindstorms EV3 platform.

## Overview

- Modeling of a two-wheeled self-balancing robot
- Analysis of the system as an inverted pendulum problem
- Open-loop and closed-loop stability analysis
- PID controller design
- Root-locus, Bode and Nyquist analysis
- MATLAB-based validation described in the report
